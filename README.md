# CafeFool - 仮想カフェのホームページ
エイプリルフールのジョークとして作成した、ちょっとユニークな仮想カフェのホームページです！

Next.jsやDockerを学ぶために作成しました。

## 使用した技術
- Next.js
- Docker
- Tailwind CSS
- ESLint
## 頑張ったとこ
- レスポンシブデザイン
- イメージスライダー
